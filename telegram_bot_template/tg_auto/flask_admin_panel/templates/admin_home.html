Инструкция по работе с сообщениями:

1. Block

Бот общается, отправляя блоки сообщений. В них может содержаться несколько сообщений разного типа (подробно о них ниже), а также несколько ответов(кнопок), которые предоставляются пользователю, после отправки этого блока

Поле Messages - сюда добавляются заранее созданные сообщения

Поле Answers - сюда добавляются заранее созданные ответы

Поле From Answer - в это поле добавляется ответ, при котором отправляется этот блок. (удобнее изменять связанное поле next_message_block в Answer)

Поле Default - в это поле добавляется заранее созданный Block, который будет отправлен, если пользователь написал ответ, которого нет в Answers. Обычно такой блок общий для всех остальных. Также в блок, который является Default для другого, не нужно добавлять ответы, тк они не отправляются

Поле Function_name - пока что оставлять пустым

2. Message

Отдельные сообщения. Имеют несколько типов, перечисленных в поле type. В случае, если type = text, поле content заполняется нужным текстом. Во всех остальных случаях, в поле content требуется указать FileID.

3. Answer

Ответы, которые может написать пользователь

Поле text - Текст ответа

Поле Button - Будет соответсвующая ответу кнопка или нет.

Поле next_message_block - Блок сообщений, который будет отправлен при этом ответе.

4. Reload

После создания всех сообщений нужно нажать на "Reload Messages". В ответ придет либо информация об успешной перезагрузке, либо код ошибки.
